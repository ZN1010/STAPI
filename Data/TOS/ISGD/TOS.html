We're only human and mistakes are occasionally made - we can reinstate disabled shortened URLs if we think it's appropriate to do so.

Excessive usage

We need to keep is.gd fast and responsive for everyone so we limit the amount it can be used to levels we consider reasonable. See our usage limits page for the current limits. If it doesn't do this and continues to hammer you may get blocked.
